## üö® Important Notes

### Before Using in Production

1. **Change ALL default passwords** immediately
2. Configure currency to your region
3. Add your real medicines to inventory
4. Register your real patients
5. Create user accounts for your team
6. Review security settings
7. Set up regular backups
8. Test thoroughly with sample data first

### Data Privacy

- All data stored locally in browser (localStorage)
- No data sent to external servers by default
- Supabase sync is optional
- Comply with local data protection laws
- Regular backups recommended

### Security

- Strong passwords required (min 6 chars)
- Role-based access enforced
- All actions logged in audit trail
- Session management active
- Regular password changes recommended
